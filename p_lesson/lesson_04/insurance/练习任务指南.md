# 练习任务指南

## 练习1: 客户流失预测（分类任务）

### 数据集信息
- 文件名: `customer_churn.csv`
- 样本数: 2000条
- 目标变量: `churn` (0=不流失, 1=流失)
- 流失率: 27.05%

### 数据特征
- `age`: 客户年龄
- `tenure`: 入网时长（月）
- `monthly_charges`: 月消费金额
- `total_charges`: 总消费金额
- `contract_type`: 合同类型
- `internet_service`: 互联网服务类型
- `online_security`: 在线安全服务
- `tech_support`: 技术支持
- `paperless_billing`: 是否无纸账单
- `payment_method`: 支付方式

### 任务目标
根据客户特征，预测客户是否会流失（二分类问题）

### 建议步骤
1. **数据探索**
   - 查看数据基本信息
   - 统计流失客户比例
   - 分析各特征与流失的关系

2. **数据预处理**
   - 处理分类变量（编码）
   - 拆分训练集和测试集

3. **模型训练**（尝试多种模型）
   - 逻辑回归
   - 决策树（建议深度=3-5）
   - 随机森林（建议100棵树）

4. **模型评估**（使用分类指标）
   - 准确率
   - 精确率
   - 召回率
   - F1分数
   - 混淆矩阵

5. **模型对比**
   - 对比三个模型的性能
   - 分析特征重要性
   - 给出业务建议

### 关键问题思考
- 哪些特征对客户流失影响最大？
- 如何降低客户流失率？
- 应该重点维护哪些客户群体？

## 练习2: 房价预测（回归任务）

### 数据集信息
- 文件名: `house_prices.csv`
- 样本数: 1500条
- 目标变量: `price` (房价，连续数值)
- 平均房价: $1,015,900
- 价格范围: $419,036 - $1,866,817

### 数据特征
- `square_feet`: 房屋面积
- `bedrooms`: 卧室数量
- `bathrooms`: 浴室数量
- `floors`: 楼层数
- `waterfront`: 是否临水
- `view`: 视野等级 (0-4)
- `condition`: 房屋状况 (1-5)
- `grade`: 房屋等级 (3-13)
- `sqft_above`: 地上面积
- `sqft_basement`: 地下室面积
- `yr_built`: 建造年份
- `yr_renovated`: 翻新年份 (0=未翻新)

### 任务目标
根据房屋特征，预测其售价（回归问题）

### 建议步骤
1. **数据探索**
   - 查看数据基本信息
   - 分析房价分布
   - 分析各特征与房价的关系

2. **数据预处理**
   - 处理缺失值（如果有）
   - 特征工程（创建新特征，如房龄）
   - 拆分训练集和测试集

3. **模型训练**（尝试多种模型）
   - 线性回归
   - 决策树回归（建议深度=4-6）
   - 随机森林回归（建议100棵树）

4. **模型评估**（使用回归指标）
   - 均方误差 (MSE)
   - 均方根误差 (RMSE)
   - 平均绝对误差 (MAE)
   - R²分数

5. **模型对比**
   - 对比三个模型的性能
   - 分析特征重要性
   - 可视化预测值vs真实值

### 关键问题思考
- 哪些因素对房价影响最大？
- 如何根据特征估算房价？
- 房屋翻新对房价有多大影响？

## 提示

### 分类任务提示
- 使用 `sklearn.linear_model.LogisticRegression`
- 使用 `sklearn.tree.DecisionTreeClassifier`
- 使用 `sklearn.ensemble.RandomForestClassifier`
- 评估指标: `accuracy_score`, `precision_score`, `recall_score`, `f1_score`

### 回归任务提示
- 使用 `sklearn.linear_model.LinearRegression`
- 使用 `sklearn.tree.DecisionTreeRegressor`
- 使用 `sklearn.ensemble.RandomForestRegressor`
- 评估指标: `mean_squared_error`, `mean_absolute_error`, `r2_score`

### 通用技巧
- 先做数据可视化，了解数据分布
- 先训练一个简单模型作为基准
- 逐步尝试更复杂的模型
- 关注特征重要性，理解模型决策
- 可视化模型结果，便于理解和汇报
